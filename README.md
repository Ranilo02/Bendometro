# Bendometro

Projeto React/Vite isolado para desenvolver o medidor de bend fora do Plectrum.

## Rodar em outro local

Depois de mover esta pasta para fora do projeto principal:

```bash
npm install
npm run dev
```

Build de producao:

```bash
npm run build
```

O componente principal fica em:

```text
src/components/BendMeter.tsx
```

Os utilitarios musicais minimos ficam em:

```text
src/utils
```
